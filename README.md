## JP-Log MVP

JP-Log는 로그인한 사용자가 방문한 일본 지역을 체크하며 기록하는 작은 여행 로그 앱입니다.

## Supabase Schema

대시보드 체크리스트를 저장하려면 아래 SQL 파일을 Supabase SQL Editor에서 한 번 실행해 주세요.

- `supabase/visited_prefectures.sql`
- `supabase/prefecture_journals.sql`
